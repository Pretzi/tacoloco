# TacoLoco

Website for **TacoLoco** — authentic Mexican street food in Kensington Market, Toronto.

- **Stack:** Next.js 15 (App Router), TypeScript, Tailwind CSS
- **Design:** Primary `#0e172a`, secondary (brand red), mobile-first, SEO-friendly
- **Sections:** Hero, Menu, Reviews/Testimonials, FAQ, Footer with address & links

## Run locally

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Build

```bash
npm run build
npm start
```

## Project structure

- `src/app/` — App Router layout, page, global styles
- `src/components/` — Header, Footer, Hero, Menu, Testimonials, FAQ, JsonLd (schema.org)
- `public/images/` — Logo and static assets
